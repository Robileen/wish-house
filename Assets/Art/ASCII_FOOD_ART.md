## 💫 Usage Instructions

### For Web Demo:
- Copy these ASCII arts into `<pre>` tags with monospace fonts
- Add CSS animations for sparkles and steam
- Use pastel colors: pink (#FFB6C1), yellow (#FFFACD), purple (#DDA0DD)

### For AI Generation:
- Use the detailed prompts below
- These ASCII arts show the composition and style
- Combine with Barbie My Scene character prompts

### For Unity/Sprites:
- Use as reference for pixel art
- Scale appropriately for game UI
- Add glow effects for magical items
