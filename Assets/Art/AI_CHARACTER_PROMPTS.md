## AI Art Generation Tips

### Best Settings:
- **Resolution**: 1024x1024 or higher
- **Style**: Digital Art, Illustration
- **Quality**: High, Detailed
- **Negative Prompts**: "blurry, low quality, distorted, ugly, bad anatomy"

### Consistency Tips:
1. **Use the same base prompt structure** for each character
2. **Keep key descriptors consistent** (hair color, style elements)
3. **Mention "Barbie My Scene style"** in every prompt
4. **Use "fashion doll proportions"** for consistent body type

### Recommended AI Tools:
- **Midjourney**: Best for artistic style
- **Stable Diffusion**: Good for consistency
- **DALL-E 3**: Great for following prompts exactly
- **NovelAI**: Good for anime-style characters

### Generation Process:
1. **Generate base character** with default expression
2. **Create variations** by changing expression and pose
3. **Maintain consistency** by using similar prompts
4. **Save in organized folders** by character name
