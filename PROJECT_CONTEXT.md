# Cafe Visual Novel - Project Context

## Game Concept
A hybrid visual novel and cafe serving game featuring 2000s Mattel Barbie My Scene aesthetics.

### Core Features
- **Visual Novel Story**: Character-driven narrative with choices
- **Cafe Serving Mechanics**: Diner Dash-style service gameplay
- **Cross-Platform**: Web (WebGL), consoles (Xbox/PlayStation/Switch), mobile

### Art Style
- **Inspiration**: 2000s Mattel Barbie My Scene characters
- **Aesthetic**: Cute cafe scenes, soft tones and shapes
- **Characters**: Fashionable, diverse cast with Barbie My Scene proportions
- **Environments**: Cozy, pastel-colored cafe interiors

## Technical Stack
- **Engine**: Unity with C#
- **Web Deployment**: WebGL via GitHub Pages
- **Platforms**: Web, consoles, mobile
